# Portfolio Website Development & Deployment Plan

## Objective
Create a modern, light-themed portfolio website as a data analyst for "Sirachut Prompt" and deploy it to GitHub Pages.

## Key Features
1. **Design:** Modern light theme with Google Sans font.
2. **Theme Toggle:** Dark mode/light mode switch button (default: light) with sun/moon icons, placed at the top right.
3. **Hero Section:** Introduction with name "Sirachut Prompt" and title "Data Analyst".
4. **Projects Section:** Showcase 3 data analysis projects.
5. **Contact Section:** Display contact details.

## Implementation Steps

### Phase 1: Structure & Styling
- Create `index.html` structure (semantic HTML5).
- Setup `css/style.css` for layout and Google Sans integration.
- Implement the Hero section.
- Create placeholder sections for Projects (3 cards) and Contact.

### Phase 2: Content
- Fill in project details (titles, descriptions, links/images).
- Populate contact information.

### Phase 3: GitHub Pages Deployment
- Initialize a local Git repository.
- Create a new GitHub repository.
- Configure repository for GitHub Pages deployment.
- Push the local project to GitHub.

## Verification
- Local testing via browser.
- Verify GitHub Pages deployment link.
- Check responsiveness across devices.
