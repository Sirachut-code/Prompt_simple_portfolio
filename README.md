# Prompt_simple_portfolio

A modern, light-themed portfolio website for a Data Analyst.

## Features
- Modern, clean design using Google Sans font.
- Responsive layout.
- Theme toggle (Light/Dark mode) with icon support.
- Portfolio showcase of 3 data analysis projects.
- Contact section.

## Technology Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Deployment
Deployed via GitHub Pages.
